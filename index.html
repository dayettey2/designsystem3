Desmond Ayettey Portfolio Design System

About this Design System

This website provides documentation for the design system used to style my Lactation resource website. This design system was created to support and display content and to help visually communicate the importance of motherhood, community, childbirth, life, and resourcefulness.

This section outlines guidelines for correct usage of elements of the typographical hierarchy, color palette, and logo set.
